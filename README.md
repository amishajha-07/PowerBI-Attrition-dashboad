# Employee Attrition Analysis Dashboard

## Project Overview
This project analyzes employee attrition data using Power BI to identify the factors that contribute to employees leaving the organization. The dashboard provides insights into attrition trends based on department, job role, age group, salary, and experience.

## Tools Used
- Power BI
- Excel / CSV Dataset
- Data Visualization

## Key Insights
- Attrition rate across different departments
- Attrition based on age groups
- Job roles with highest employee turnover
- Salary and experience impact on attrition
- Gender-wise attrition comparison

## Dashboard Features
- Interactive filters for department and job role
- Visual charts showing attrition trends
- KPI cards showing total employees and attrition rate
- Data-driven insights for HR decision making

## Dataset
The dataset contains information about employees such as:
- Age
- Department
- Job Role
- Monthly Income
- Years at Company
- Attrition Status

## Project Files
- `Attrition_Dashboard.pbix` – Power BI dashboard file
- `HR_Attrition_Dataset.csv` – Dataset used for analysis
- `dashboard.png` – Dashboard preview image

## Objective
The goal of this project is to help HR teams understand the key reasons behind employee attrition and support better workforce planning using data analytics.
